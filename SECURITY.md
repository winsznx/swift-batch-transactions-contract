# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security vulnerabilities very seriously. If you discover a security issue, please bring it to our attention right away!

### How to Report

Please do **NOT** file a public issue. Instead, please send an email to security@swiftv2.com (or equivalent contact method), or send a DM to one of the maintainers.

### What to Include

*   Documentation of the vulnerability.
*   A Proof of Concept (PoC) or instructions to reproduce the issue.
*   Impact of the vulnerability.

### Response

The team will acknowledge receipt of your report within 48 hours and will send you a detailed response with an estimated timeline for a fix.
